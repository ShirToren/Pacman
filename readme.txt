Colors are implemented.

We used file system so you need to change project to C++ 17.

File's format:

when a fruit is appearing we wrote
F Appearance 5 14 69
when 5 is the digit and (14,69) is the first position of the fruit.
when a fruit is disappearing we wrote 
F DisAppearance

for every step of the pacman we wrote
P A
when 'A' is for the direction (left).

for every step of one ghost we wrote
G A

and for the fruit is the same
F A
